# Plugin Assets for WordPress.org

This folder contains assets for the WordPress.org plugin directory.

## Required Files

### Screenshots (for readme.txt)
Screenshots should be named `screenshot-N.png` where N is the screenshot number.

- `screenshot-1.png` - Dashboard overview with connected platforms
- `screenshot-2.png` - AI Assistant chat interface
- `screenshot-3.png` - Google Analytics 4 connection setup
- `screenshot-4.png` - Settings page with API configuration

Recommended size: 1200x900 pixels (or similar 4:3 ratio)

### Plugin Banner
- `banner-772x250.png` - Low resolution banner (required)
- `banner-1544x500.png` - High resolution banner (recommended)

### Plugin Icon
- `icon-128x128.png` - Low resolution icon (required)
- `icon-256x256.png` - High resolution icon (recommended)

## Notes

- All images should be PNG format
- Use descriptive alt text in readme.txt for screenshots
- Keep file sizes optimized (compress PNGs)
- Banner should represent the plugin's purpose clearly
- Icon should be recognizable at small sizes
